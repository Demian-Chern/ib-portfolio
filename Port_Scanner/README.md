# Port Scanner

Набор инструментов для сканирования открытых портов на Python.

Два варианта:
- Простая версия (для понимания основ)
- Продвинутая версия с логированием, argparse и проверками

---


### Возможности

**Skan_v1.py (основная версия):**
- Сканирование по списку портов
- Удобный CLI-интерфейс (`argparse`)
- Логирование в файл и консоль
- Проверка на "фейковые" открытые порты (защита от провайдерских заглушек)
- Таймауты и обработка ошибок
- Красивый вывод результатов

**port_scan.py** — простая учебная версия (для понимания базового механизма).

---

##  Как использовать

### Основная версия (рекомендуется)

```bash
python Skan_v1.py --host example.com --ports 21,22,80,443,8080 --timeout 1.0
```
Аргументы:

--host — целевой хост (IP или домен) — обязательный
--ports — порты через запятую (по умолчанию: 21,22,80,443)
--timeout — таймаут в секундах (по умолчанию: 1.0)

### Пример
```bash
Bashpython Skan_v1.py --host online.mospolytech.ru --ports 80,443,22,3389
```
